１．調整項目の追加について

　　1)　エディタ−を使用して、「ALL_ITEM.TXT」を呼び出して下さい。
　　　　（エディタ−は、MS-DOSに付随しているもの、市販されているもの、
　　　　　何でも結構です。)

　　2)　現在登録されている最終項目の次から新規に追加する項目を入力して下さい。

　　　　ａ．登録できる最大数は、９９件までです。
　　　　ｂ．１レコ−ドは、６７バイトで構成されています。
        ｃ．１レコ−ドの構成は、以下のようになっています。

    1                                                                 67
    |-----------------------------------------------------------------|
　　0101APC                       [  ][  ][  ][  ][  ]  021c00100ff208c[Enter]
　　0201OUT_BIAS                  [  ][  ][  ][  ][  ]  021a00100ff208a[Enter]
　　0301HUE                       [  ][  ][  ][  ][  ]  021900100ff2089[Enter]
　　　　　　　　　　　　　　　　　　・
　　　　　　　　　　　　　　　　　　・
　　　　　　　　　　　　　　　　　　・
　　1501SYS_PHASE(1-1)            [  ][  ][  ][  ][  ]  1e0012112130000[Enter]
　　1502SYS_PHASE(1-2)            [  ][  ][  ][  ][  ]  1e0014000000000[Enter]
　　1601SYS_PHASE(2-1)            [  ][  ][  ][  ][  ]  1e0015115180000[Enter]
　　1602SYS_PHASE(2-2)            [  ][  ][  ][  ][  ]  1e0019000000000[Enter]
　　1701SYS_PHASE(3-1)            [  ][  ][  ][  ][  ]  1e0019000000000[Enter]
　　1702SYS_PHASE(3-2)            [  ][  ][  ][  ][  ]  1e001A11a1b0000[Enter]
　　1703SYS_PHASE(3-3)            [  ][  ][  ][  ][  ]  1e001C000000000[Enter]

          |-------------------------------------------------------------|
　  　　　|　　　　１〜　２文字目までの　２文字　　大項目番号           |
　  　　　|　　　　３〜　４文字目までの　２文字　　小項目番号           |
　　　　　|　　　　５〜５２文字目までの４８文字　　調整名称（英語）     |
　　　　　|　　　５３〜５４文字目までの　２文字　　書き込みコマンド     |
　　　　　|　　　５５〜５６文字目までの　２文字　　書き込みアドレス     |
　　　　　|　　　５７〜５８文字目までの　２文字　　書き込みデータ       |
　　　　　|　　　　　　５９文字目の　　　１文字　　判別フラグ           |
　　　　　|　　　６０〜６１文字目までの　２文字　　書き込み開始データ   |
　　　　　|　　　６２〜６３文字目までの　２文字　　書き込み終了データ   |
　　　　　|　　　６４〜６５文字目までの　２文字　　読み込みコマンド     |
　　　　　|　　　６６〜６７文字目までの　２文字　　読み込みアドレス     |
          |-------------------------------------------------------------|


　　　　ｄ．上記の内容に従い、新規の項目を入力して下さい。
　　　　ｅ．なお、１項目（１レコード）の最後（６８バイト目）には、必ず、
　　　　　　［Ｅｎｔｅｒ］キーを押して下さい。
　　　　　　［Ｅｎｔｅｒ］キーを入力しなければ、正常な登録は行えません。


２．調整メニューの追加について

　　1)　エディタ−を使用して、「ADJ_MENU.TXT」を呼び出して下さい。
　　　　（エディタ−は、MS-DOSに付随しているもの、市販されているもの、
　　　　　何でも結構です。)

　　2)　現在登録されている最終項目の次から新規に追加する項目を入力して下さい。

　　　　ａ．登録できる最大数は、１５件までです。

          |-------------------------------------------------------------|
　　　　　|　　　VIDEO ADJUSTMENT PROCESS. < NTSC >[Enter]              |
　　　　　|　　　CAMERA ADJUSTMENT PROCESS.< NTSC >[Enter]              |
　　　　　|　　　VIDEO ADJUSTMENT PROCESS. < PAL >[Enter]               |
　　　　　|　　　CAMERA ADJUSTMENT PROCESS.< PAL >[Enter]               |
　　　　　|　　　ここから追加して下さい。                               |
          |-------------------------------------------------------------|

　　　　ｂ．なお、１項目の最後には、必ず、［Ｅｎｔｅｒ］キーを押して下さい。
　　　　　　［Ｅｎｔｅｒ］キーを入力しなければ、正常な登録は行えません。
